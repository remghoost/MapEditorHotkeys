## Mod for Ravenfield.

**Adds hotkeys to the Map Editor.**  
**Now with configurable hotkeys!**

Hotkeys are as follows:

- Translate = V
- Rotate = R
- Scale = F
- Terrain = T 
- Place = G

It shouldn't conflict with anything, but who knows. lmao.  
Don't try to break the mod, please. <3

---
### **Requires BepInEx 5.**

### If you already have it:  
1. Download `.dll` file from the ["Releases" section on the right](https://github.com/remghoost/MapEditorHotkeys/releases/).
2. Extract into `Ravenfield\BepInEx\plugins`.

### If you don't already have it:
1. Download the zip file from the ["Releases" section on the right](https://github.com/remghoost/MapEditorHotkeys/releases/).
2. Extract the entire zip file into `Ravenfield` (where `ravenfield.exe` lives).
3. Start game once then close it to make sure BepInEx initializes properly.
